🍕 SmartSlice – Pizza Delivery Web Application

🚀 SmartSlice is a modern and interactive Pizza Delivery Web Application that simulates a real online food ordering system. This project allows users to browse pizza varieties, customize their pizza, add items to the cart, and place orders through a smooth and user friendly interface.

This project was developed as part of my Web Development Internship at Oasis Infobyte, where I worked on building practical and real world web applications to strengthen my frontend development skills.

---

✨ Features

• 🔐 User Login and Registration system
• 🍕 Browse different pizza varieties
• 🧀 Customize pizza (size, sauce, cheese, toppings)
• 🛒 Add pizzas to cart and manage orders
• 💳 Checkout with payment simulation
• 📦 Order tracking system with delivery stages
• 📊 Admin dashboard to view and manage orders
• 🧾 Inventory management for pizza ingredients
• 📱 Fully responsive and modern UI design

---

🛠 Technologies Used

• CSS

These technologies were used to create a responsive and interactive web application with smooth functionality and clean design.
